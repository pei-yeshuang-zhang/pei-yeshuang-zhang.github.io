Differences Between Margin Padding and Border

Box Model

If you're new to coding, like I am, the box model is one of the most important things to understand in CSS. The CSS box model consists of 4 main areas: (from outermost to innermost) the margin, border, padding, and the content section. Each of these 4 areas of the box model in CSS has a different purpose and can be manipulated with CSS code.

Border

The border in the CSS box model is the area between the margin and the padding of an element. It can be set to have a different width, style, and color for each side. The border element of the box model can be used to create a border around an element or to give it more definition.

Margin

Margin is what creates the gap around the content block, by “pushing” the content away. It's always outside of the border. The margin can be set for each side of an element and can be a different size for each side.

When to use margin:

  • To create a gap/space around the content block.
  • To make a space between two elements.

Padding

Padding is the space between the content of an element and its border. Just like margin, padding can be applied to the top, bottom, left, and right sides of an element, and it can be different for each side.

When to use padding:

  • To create a gap between the edges of the container and the content in it.
  • You want to increase the size of the content block, or create a bigger space within the border.
  • Increase the size of the content block without making the content itself bigger.

The main differences between margin and padding:

  • Padding alter dimensions. Margin does not.
  • Margins can be negative. Padding cannot.
  • Margin can be auto. Padding cannot.
